# ZeroForm FOC Pro

## 正式发布

**ZeroForm FOC Pro** 由 **芯辰科技（HW）** 与 **ZeroForm Studio（SW）** 联合研发，致力于打造功能完善、成本友好、面向研究与教育场景的 FOC 驱动解决方案。

---

## 硬件特性

- **主控 MCU**：STM32G431（最高支持 20kHz 电流环）
- **电流采样**：INA240A1 高精度电流检测（相线检测方案）
- **位置反馈**：MT6835 21 位磁编码器，高分辨率位置采集
- **通信接口**
    - 串口通信
    - FDCAN（5 Mbps，验证中）
    - USB CDC（开发中）

---

## 软件特性

- 完整 FOC 算法实现
- 支持高速电流环与高带宽控制
- 模块化代码结构设计
- 文档完善，快速上手
- 从硬件到软件 **100% 开源**
